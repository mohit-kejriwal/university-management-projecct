
UNIVERSITY MANAGEMENT SYSTEM

Efficiently manage university operations with role-based dashboards.
Add and manage students and faculty, track attendance, analyze performance, and communicate seamlessly.

--------------------------
ABOUT THE PROJECT
--------------------------

The University Management System is a web-based application built using the MERN stack:
MongoDB, Express.js, React (with Vite), and Node.js.

This system simplifies academic and administrative workflows, offering tailored interfaces
and functionalities for Admins, Teachers, and Students.

--------------------------
FEATURES
--------------------------

- Role-Based Access:
  Admin, Teacher, and Student dashboards with unique access and permissions.

- Admin Capabilities:
  Add and manage students and teachers, assign subjects, manage classes,
  view reports, and export data.

- Teacher Portal:
  Take and manage attendance, assign marks and feedback, view student progress,
  and send messages.

- Student Dashboard:
  View profile, attendance records, marks, feedback, and performance analytics.

- Data Visualization:
  View performance via pie charts and tables.

- File Export:
  Export student records and attendance data to Excel/CSV.

--------------------------
TECH STACK
--------------------------

Frontend:
- React.js with Vite
- Material UI
- Redux Toolkit
- Axios

Backend:
- Node.js
- Express.js
- MongoDB
- JWT (Authentication)

--------------------------
INSTALLATION INSTRUCTIONS
--------------------------

1. Clone the repository:

https://github.com/mohit-kejriwal/university-management-projecct.git   cd University-management-app

2. Open two terminal windows:

--- Terminal 1 (Backend) ---

   cd backend
   npm install
   npm start

--- Terminal 2 (Frontend) ---

   cd frontend
   npm install
   npm run dev

3. Visit the application in your browser:

   Frontend: http://localhost:5173
   Backend API: http://localhost:5000
